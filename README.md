# Fast portscanner
Usage
```python
python3 scanner.py <ip> <debugging> <socket timeout>
```
## ARGS
|                |Default                        |What can it be               |
|----------------|-------------------------------|-----------------------------|
|IP              |No default value               |any ipv4 adress              |
|Debugging       |No                             |Y or anything else           |
|Timeout         |2.5                            |Any float number             |
